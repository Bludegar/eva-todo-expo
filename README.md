# eva-todo-expo

proyecto de ejemplo: todo list con expo, typescript y expo-router

instrucciones basicas:

- instalar dependencias:

```bash
npm install expo-router @react-native-async-storage/async-storage expo-file-system expo-image-picker expo-location uuid
```

- ejecutar en desarrollo:

```bash
npm start
```

notas:
- incluye login simple (sin backend) y tareas guardadas en AsyncStorage por usuario.
- las fotos se guardan en el filesystem local del dispositivo.
